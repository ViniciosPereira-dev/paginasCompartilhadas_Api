#📚 Paginas Compartilhadas API

🎯 Contexto e Problema
O acesso à leitura ainda é limitado para muitas pessoas, seja por questões financeiras ou pela falta de disponibilidade de livros em determinadas regiões. Ao mesmo tempo, muitos leitores possuem livros parados em casa que poderiam ser reutilizados.
O problema que buscamos resolver é justamente essa desconexão entre quem tem livros para doar e quem deseja recebê-los.
